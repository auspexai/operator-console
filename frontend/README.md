# AuspexAI Operator Console — frontend

The SvelteKit + TypeScript frontend for the AuspexAI operator console. See [`../README.md`](../README.md) for what the console does, how it fits the coordinator, and how to run the backend.

## Development

```sh
npm install       # install dependencies
npm run dev       # start the dev server
npm run build     # build the production bundle
```
